大驼峰 例：UserName MyDocument

小驼峰 例：userName passWd

方法的调用：直接写出即可，如new方法调用时写出 new();

在方法中声明的变量成为形参，而在方法中传递的变量成为实参

在调用方法时，要传递进去一个“实际的”变量，这个变量就叫做实参。
